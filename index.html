The Basics

Those with experience in HTML will have seen many anchor tags, which are written as <a></a>. For those without experience, this is the simplest version of the code we use to create links on a webpage.

The anchor tag can be used to create several different kinds of links:

  • External, taking you to a different webpage
  • Internal, helping you jump to a specific section of a longer page
  • Email links, which open your default email app
  • Phone links, which will open on your mobile phone
  • There are other things you can do with an anchor tag, but these are what we will be covering in this documentation.

You can make text into a link by putting the opening and closing anchor tags around it, then adding the further link information covered in later sections. The most basic example is this: <a>your text here</a>. (This example won't take you anywhere! It's just the start of structuring the link.)

href and URLs

Most anchor tags are used to take you to a different webpage, either within your site or to an entirely different site. Do do this, you use a Hypertext Reference (href) to link the desired URL. The proper way to build a link is <a href="https://www.example.com">This is an example!</a>

Make note of the structure -- opening angle bracket, a for anchor, then href for hypertext reference, with an equals sign and quotation marks around the target URL, the the closing angle bracket. Second, your link text. Third, opening and closing angle brackets around a slash / and a second a for anchor.

Also note that this structure (<tag attribute="value"></tag>) is common for several HTML tags and is worth getting used to!

Within a Page

When you want to link within a page, you can use an id attribute in your tag and referring to it in your anchor tag, which is generally in your navigation bar or (in general) in a different part of a larger page.

The structure of the anchor tag is <a href="#smile">Smile!</a> and the structure of the target is <tag id="smile">Smile!</tag>. The pound/hashtag symbol in the anchor tag's value tells the computer that it's looking for an id. (This format is also used in CSS when referring to an id.)

Email Links

When you want the viewer to open their default email app with your email address pre-filled, you can also use an anchor tag.

The structure of the anchor tag for email is <a href="mailto:email@example.com">Send email</a>. You can also pre-fill a subject line using the following structure: <a href="mailto:email@example.com?subject=Support Request">Send support request email</a>.

Phone Links

Finally we come to phone links. These anchor tags use tel: before the number. Note that browser support for linking telephone numbers can vary.

The structure of the anchor tag for telephone numbers is <a href="tel:+15555555555">1-555-555-5555</a>. On devices that are capable of making phone calls, this is intended to initiate a call when the link is clicked.
